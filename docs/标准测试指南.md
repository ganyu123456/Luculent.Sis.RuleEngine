## 问题排查

### Master 日志显示 "从 Monitor Center 拉取完成: 0 个监视项"

1. 检查 MonitorCenter 是否正常运行: `docker ps | grep sis-service`
2. 检查 PostgreSQL 中是否有监视项:
   ```sql
   SELECT count(*) FROM ssmcitemmst WHERE valid_sta = 'A';
   ```
3. 检查 MonitorCenter API: `curl -s "http://127.0.0.1:11080/api/services/monitorcenter/monitorDataForPublic/GetAllMonitors"`
4. 检查 Master 日志中的 API 错误

### Worker 反复断连重连

1. 检查 gRPC 消息大小配置 (Master Program.cs + Worker GrpcConnectionService.cs)
2. 检查 Master 健康: `curl http://localhost:11082/api/ruleengine/health`
3. 检查 Docker 网络: `docker network inspect 03-luculentsisruleengine_ruleengine`

### ClickHouse 无新事件

1. 检查 TrendDB 数据源: `db05.test1` 是否有实时值
2. 检查 Worker 数据采集: `docker logs worker-1 | grep "数据采集"`
3. 检查 Worker 计算: `docker logs worker-1 | grep "计算"`

### Redis 无活跃报警

1. 检查监视项配置: `prerule_no` 是否正确关联
2. 检查前置规则条件是否过于严格
3. 检查 Redis 连接: `docker exec ruleengine-redis redis-cli PING`

## 常用 SQL 查询 (ClickHouse)

```sql
-- 最近 N 分钟事件统计
SELECT 
    count() as total,
    countIf(status_key = 'satisfiled') as triggers,
    countIf(status_key = '') as clears,
    uniqExact(monitor_id) as active_monitors
FROM ruleengine.alarm_events
WHERE occur_time >= now() - INTERVAL 10 MINUTE;

-- 单监视项事件历史
SELECT occur_time, status_key, trigger_value, worker_id
FROM ruleengine.alarm_events
WHERE monitor_id = 'MON0000001'
ORDER BY occur_time DESC
LIMIT 10;

-- Worker 负载分布
SELECT worker_id, count() as cnt, uniqExact(monitor_id) as monitors
FROM ruleengine.alarm_events
WHERE occur_time >= now() - INTERVAL 5 MINUTE
GROUP BY worker_id;

-- 检查未关闭事件
SELECT monitor_id, status_key, count() as cnt
FROM ruleengine.alarm_events
WHERE occur_time >= now() - INTERVAL 1 HOUR
GROUP BY monitor_id, status_key
HAVING cnt % 2 != 0
LIMIT 10;
```

## 常用 SQL 查询 (PostgreSQL)

```sql
-- 监视项统计
SELECT 
    COUNT(*) as total_monitors,
    COUNT(DISTINCT prerule_no) as prerules
FROM ssmcitemmst WHERE valid_sta = 'A';

-- 前置规则关联监视项分布
SELECT prerule_no, COUNT(*) as cnt
FROM ssmcitemmst WHERE valid_sta = 'A'
GROUP BY prerule_no;

-- 数据源类型分布
SELECT source_flag, COUNT(*) as cnt
FROM ssmcsourcemst WHERE valid_sta = 'A'
GROUP BY source_flag;
```
